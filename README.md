# Library Management System (SQL)

## Project Overview

This project demonstrates the design and analysis of a Library Management System using MySQL. The database manages books, members, employees, book issuance, returns, overdue tracking, and branch-level reporting.

## Database Components

* Books
* Members
* Employees
* Branches
* Issued Status
* Return Status

* customers
* deliveries
* orders
* restaurants
* riders 

## SQL Skills Demonstrated

* Database Design
* CRUD Operations
* Joins (INNER, LEFT)
* Aggregate Functions
* GROUP BY & HAVING
* Date Functions
* Common Table Expressions (CTEs)
* Stored Procedures
* Business Reporting Queries

## Key Business Problems Solved

* Identify overdue books
* Calculate overdue fines
* Track book availability
* Find active library members
* Analyze branch performance
* Monitor damaged book returns
* Identify top-performing employees
* Generate rental revenue reports

## Tools Used

* MySQL
* MySQL Workbench
* GitHub
